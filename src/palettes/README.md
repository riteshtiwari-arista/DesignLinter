# Design System Palettes

This directory contains extracted design system palettes.

## Adding a New Palette

1. Extract palette from your DS file (see `/EXTRACT_PALETTE.md`)
2. Save JSON as `[ds-name].json` in this directory
3. Import it in `index.ts`
4. Map it to a library key

## File Naming

- Use kebab-case: `geiger-design-system.json`
- Match the library name in Figma
- One file per design system

## Current Palettes

- `geiger-design-system.json` - Geiger Design System (placeholder)

To extract your actual palette, run the extractor script in your DS file.
